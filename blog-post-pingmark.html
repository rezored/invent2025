Pingmark: Moving beyond GPS coordinates

GPS gives us a 'where', but it lacks the 'what'. Pingmark is an open protocol designed to attach semantic meaning to spatial coordinates, creating a persistent augmented reality layer for the web.

The Semantic Gap

Current location services tell you that you are at 40.7128° N, 74.0060° W. They don't tell you that this is a "safe drop-off zone" for a drone, or a "historic preservation site" where AR ads should be disabled.

The Protocol

Pingmark uses a decentralized registry to map geohashes to semantic tags. 

interface PingmarkNode {
  geohash: string; // "dr5ru7"
  semantics: {
    type: "infrastructure" | "commercial" | "restricted";
    metadata: Record;
    ttl?: number;
  }
}

This allows autonomous agents and AR devices to query their environment for context, not just coordinates. We are currently piloting this with academic partners for campus navigation.
